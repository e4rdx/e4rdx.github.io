# [Version 1.2](https://github.com/e4rdx/sNote/releases/tag/1.2)
Download [snote 1.2]({{ site.url }}/download/snote/sNote1.2.apk)\\
Changes in this version:
* Just click on an checklist entry to start editing it
* Tags can be added now to checklist, tag system will be expandet to other note types in next update!
* When a new entry is added to the checklist, it scrolls down to the new entry
* Link support: Put a link in a checklist and it will be marked, long click on an entry with link to open it!
* New experimental feature: External notebooks - Use a .snote file anywhere in your storage as notebook! Activate this feature in the settings, but be carful, there are still some bad bugs...
